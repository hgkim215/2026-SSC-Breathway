# 호흡재활 Gamification 아이디어 정리 (SSC용)

## 1. 한 줄 컨셉

`Breath Quest`: 호흡기 환자가 매일 1~2분짜리 호흡재활 미션을 게임처럼 수행하며, 부담 없이 지속할 수 있게 돕는 오프라인 중심 앱

## 2. 해결하려는 핵심 문제

- 호흡재활 운동은 중요하지만 지루함, 불안, 피로로 인해 지속하기 어려움
- 사용자는 운동법을 "알아도" 매일 실천하지 못함
- 보호자는 환자의 수행 여부와 상태 변화를 직관적으로 확인하기 어려움

## 3. 제품 가설

- 짧은 미션 + 즉시 보상 + 연속 수행 시각화가 재활 지속률을 높인다
- "완벽한 수행"보다 "오늘도 했다"를 강화하면 이탈이 줄어든다
- 위기 신호 체크를 함께 넣으면 안전성과 신뢰도가 올라간다

## 4. Gamification 설계

## 4-1. 핵심 루프

`오늘의 미션 시작 -> 미션 수행(60~120초) -> 즉시 보상(XP/배지) -> 주간 성장 확인 -> 내일 재도전`

## 4-2. 보상 구조

- XP: 미션 완료 시 기본 지급
- Streak: 연속 수행 일수 표시
- Badge: 3일/7일/14일 달성 배지
- Recovery Bonus: 컨디션이 안 좋은 날 "라이트 미션" 완료 시 별도 보상

## 4-3. 난이도 원칙

- Day 1~3: 초저강도 적응
- Day 4~7: 리듬/호흡 길이 소폭 증가
- 사용자 상태가 나쁘면 자동으로 쉬운 미션 제안

## 5. 미션 예시

- `Paced Breathing`: 4초 들숨, 6초 날숨 리듬 따라가기
- `Pursed-Lip Challenge`: 오므린 입술 호흡으로 천천히 내쉬기
- `Posture Boost`: 60초 자세 교정 + 어깨/흉곽 이완
- `Mini Move`: 의자 기반 저강도 1분 움직임

## 6. 안전 가드레일 (필수)

- 시작 전/후 자가 체크(숨참 정도, 어지러움 여부)
- 위험 신호(흉통, 심한 호흡곤란, 어지러움) 선택 시 즉시 중단 안내
- 의료 고지 문구: 본 앱은 의료진 대체가 아님, 응급 시 지역 응급번호 사용

## 7. SSC 3분 데모 구성

- 0:00~0:30 문제와 사용자 소개
- 0:30~2:00 미션 1개 실시간 플레이
- 2:00~2:40 보상/성장 화면(배지, streak, 주간 캘린더)
- 2:40~3:00 접근성/안전 가드레일 강조

## 8. MVP 범위 (2주)

- In Scope:
- 오늘의 미션 3종
- XP/연속일/배지 시스템
- 기본 안전 체크 플로우
- 오프라인 로컬 데이터 저장

- Out of Scope:
- 병원 시스템 연동
- 진단/치료 자동 판단
- 복잡한 소셜 기능

## 9. 성공 지표 (초안)

- North Star: 주 5일 이상 미션 완료 사용자 비율
- Leading: 1회 미션 완료율, 7일 유지율, 평균 세션 길이
- Guardrail: 사용 후 불안도 악화(자가보고) 비율 10% 이하
