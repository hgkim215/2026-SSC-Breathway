## 2. 최종 선정: Top 3 핵심 아이디어

### 🥇 1순위: **침묵 속의 SOS (Silent SOS)**

**Why?**

- 가장 핵심적인 문제 해결: "말할 힘이 없어도 도움 요청"
- 혁신성: 비음성 긴급 호출은 기존 앱에 없는 독창적 접근
- 사회적 영향: 생명을 구할 수 있는 직접적 가치
- 3분 데모 perfect: 시연이 명확하고 감동적

**핵심 기능:**

- 말하지 않고 긴급 호출 (눈 깜빡임, 머리 움직임, 탭 등)
- Apple Watch 제스처/패턴 인식
- 즉시 보호자/가족에게 알림 + 위치 공유

### 🥈 2순위: **위기 대응 가이드 (Crisis Guide)**

**Why?**

- 보호자의 가장 큰 고통 해결: "지금 뭘 해야 하나?"
- 실현 가능성 매우 높음: 4주 안에 완성 가능
- 사회적 영향: 보호자 PTSD와 판단 마비 해소
- 포용성: 의료 지식 없는 보호자도 사용 가능

**핵심 기능:**

- 증상 체크리스트 → 대응 방법 안내
- "응급실 가세요" vs "자세 바꿔보세요" 판단 도움
- 단계별 음성/시각 가이드
- 호흡 곤란 대처법 인터랙티브 튜토리얼

### 🥉 3순위: **보호자 대시보드 (Caregiver Dashboard)**

**Why?**

- 보호자의 24시간 불안 해소: "평화로운 침묵 vs 위험한 침묵"
- Apple 생태계와 완벽 통합 (HealthKit, Apple Watch 데이터 공유)
- 3분 데모 가능: 환자-보호자 연결 시연
- 이중 고통 해결: 환자뿐 아니라 보호자도 돌봄

**핵심 기능:**

- 환자의 실시간 상태 모니터링 (호흡, 심박, 활동)
- "평화로운 수면" vs "호흡 곤란" 자동 구분
- 위급 상황 즉시 알림
- 트렌드 그래프 (증상 악화 조기 발견)

## 4. Must-Have vs Nice-to-Have

### Must-Have (3분 데모에 필수)

**환자용:**

1. ✅ **Silent SOS**: 비음성 긴급 호출 (탭, 제스처)
2. ✅ **간단한 메시징**: "물", "화장실", "숨 차" 미리 저장 메시지

**보호자용:** 3. ✅ **실시간 알림**: 환자 SOS 시 즉시 알림 4. ✅ **위기 대응 가이드**: "지금 뭘 해야 하나요?" 단계별 안내 5. ✅ **상태 대시보드**: 환자 현재 상태 한눈에

### Nice-to-Have (시간 여유 있으면 추가)

**환자용:**

- AI 호흡 패턴 모니터링 (Core ML - 복잡)
- 근육 게임화 동기부여
- 안전한 산책 코치

**보호자용:**

- 트렌드 분석 그래프
- 의사 리포트 자동 생성
- 보호자 명상/커뮤니티

**공통:**

- 산소줄 꼬임 감지
- 산소통 잔량 예측

> 거실에 있던 저는 아무것도 몰랐고,

---

### [0:45 - 2:15] 솔루션 시연 - 핵심 기능

**Scene 1: Silent SOS (30초)**

- 환자 화면: Apple Watch에서 두 번 탭 → "SOS 전송됨"
- 보호자 화면: 즉시 알림 + "환자가 도움을 요청합니다" + 위치 표시
- 내레이션: "말하지 않아도, 두 번 탭만으로 도움을 요청할 수 있습니다."

**Scene 2: 위기 대응 가이드 (45초)**

- 보호자가 환자 방으로 달려감
- 앱 화면: "호흡 곤란 대처 가이드" 자동 표시
  - Step 1: "환자를 앉게 하세요" ✓
  - Step 2: "상체를 30도 세우세요" ✓
  - Step 3: "천천히 호흡하도록 유도하세요"
  - AI 판단: "증상이 10분 내 개선되지 않으면 119에 연락하세요"
- 내레이션: "보호자는 당황하지 않고, 정확히 대처할 수 있습니다."

**Scene 3: 보호자 대시보드 (30초)**

- 대시보드 화면:
  - 현재 상태: "안정" (녹색)
  - 호흡수: 18회/분
  - 심박수: 78bpm
  - 산소포화도: 94%
  - 트렌드: "지난 주 대비 개선"
- 내레이션: "이제 보호자는 안심하고 잘 수 있습니다."

## 7. 기술 스택 (예상)

### 필수 기술

- **SwiftUI**: 현대적 UI
- **HealthKit**: 호흡수, 심박수, 산소포화도
- **WatchConnectivity**: Apple Watch ↔ iPhone 통신
- **UserNotifications**: 긴급 알림
- **Core Motion**: 제스처 인식 (Watch)

### 선택 기술

- **Core ML**: AI 위험도 판단 (시간 여유 있으면)
- **Swift Charts**: 트렌드 그래프
- **WidgetKit**: 보호자용 위젯
- **SharePlay / iCloud**: 보호자-환자 데이터 공유
